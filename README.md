# 🩸 Blood Donor Directory

A web-based platform to add and search blood donors.

## 🚀 Features
- Add donor details
- Search donors by blood group & district
- Firebase Firestore database integration
- Deployed using Firebase Hosting

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Firebase
